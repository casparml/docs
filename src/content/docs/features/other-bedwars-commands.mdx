## Description

The following commands can be used to view other Bedwars stats.

## Stat Commands

import CommandInfo from '../../../components/CommandInfo.jsx';

### Current Bedwars Stats

<CommandInfo
    commandSyntax="/bedwars [<player>]"
    commandDescription="Show the current general Bedwars stats for a player."
    args={[
        {
            "name": "player",
            "description": "The player to view Bedwars stats for."
        }
    ]}
/>

### Stat Comparison

<CommandInfo
    commandSyntax="/compare <player_1> [<player_2>]"
    commandDescription="Show the stat comparison between two players."
    note="Not specifying the second player will compare to the first player to your linked player."
    args={[
        {
            "name": "player_1",
            "description": "The first player to compare."
        },
        {
            "name": "player_2",
            "description": "The second player to compare."
        }
    ]}
/>

### Stat Milestones

<CommandInfo
    commandSyntax="/milestones [<player>] [<session>]"
    commandDescription="Show the next stat milestones for a player."
    note="Session stats will be factored into the calculations. To not use session stats, specify a session ID of 0."
    args={[
        {
            "name": "player",
            "description": "The player to view stat milestones for."
        },
        {
            "name": "session",
            "description": "The session ID of the session to derive milestones from."
        }
    ]}
/>
